= API calls

The connector does the following API calls during the sellers extraction jobs execution:

|===
| Call | Target System | Endpoint | Parameters

| Confirms the payment of a seller billing cycle setting its status to `PAID`.
| Mirakl
| https://developer.mirakl.com/content/product/mms/rest/front/openapi3/invoicing-and-accounting/sbc07[PUT SBC07 - Confirm the payment of seller billing cycle]
a| * `id`: seller billing cycle identifier
* `currency_iso_code`: ISO code for the currency used in the payment.
* `amount_transferred_to_seller`: Amount of the invoice.
* `transaction_date`: Invoice transaction date.
* `state`: Payment state of the invoice.
|===
